# Inkwell — Frontend

React + TypeScript + Vite SPA for the Inkwell blogging platform.

See the root [`README.md`](../README.md) for the full project overview, and
[`../docs/DEVELOPER_GUIDE.md`](../docs/DEVELOPER_GUIDE.md) for local setup instructions.

Quick start:

```bash
cp .env.example .env
npm install
npm run dev
```
